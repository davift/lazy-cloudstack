# Lazy CloudStack

Quick deployments of Apache CloudStack 4.22 on Ubuntu 24.04 LTS for local tests.

# Ubuntu 24.04 LTS

curl -s https://raw.githubusercontent.com/davift/lazy-cloudstack/refs/heads/main/u24-install.sh | sudo bash

# Uninstall

curl -s https://raw.githubusercontent.com/davift/lazy-cloudstack/refs/heads/main/u24-reinstall.sh | sudo bash
